Fuzzy Mapper

The Fuzzy Mapper is a simple Windows Forms application that will allow you to compare & map data from 2 excels using fuzzy string matching algorithms. The application allows you to specify the columns which need to be compared (one or more) & also allows you to specify the accuracy at which the algorithms should operate. The code & sample app can be found here.

Features:

  • Supports fuzzy matching using 3 algorithms - Levenshtein Distance, Dice Coefficient & Longest Common Subsequence
  • Allows users to specify accuracy at which alogrithms are applied
  • Mapping can be performed based on multiple columns at once
  • Data from 2 excels can either be Mapped or Merged
  • Intuitive & easy to use UI that includes grid features like sorting, filtering, column chooser & column grouping
  • Data can easily be exported to excel

File Comparison

A simple Python script that compares the contents of 2 folders and spits out an excel with the results. The script iteratively scans through all sub folders and files of the source and searches for the corresponding file & folder structure in the destination. For each file it finds it performs an MD5 checksum comparison to verify if they are the same. The code can be found here.

Features:

  • Light weight & easy to use
  • Extremely fast, kicks off multiple threads to process the files asynchronously
  • Will iterate through all sub folders and files
  • Computes the total size of all the files found at source and all corresponding files fount at destination
  • Computes an MD5 checksum of both source and destination file for true comparison
  • Exports an excel with all the details

Tech Stack: Python 3.6

Content LookUp

An angular application to lookup contents from the Library of Congress & EIDR. The code can be found here.

Features:

  • Reflects how to implement a simple search as you type functionality using rxjs observables, subject and debounceTime & distinctUntilChanged operators
  • The response from EIDR is an XML response which is converted to JSON using the xml2js package
